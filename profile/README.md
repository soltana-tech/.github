# soltana-tech

Open-source tools for academic and personal scripture study. The focus is on software that is free to use, built in the open, and built to last.

---

## The Problem

Free, serious study tools for religious texts are scarce. Tools with academic depth — original languages, morphology, cross-references, comparative traditions — either cost money (Logos, Accordance) or lack breadth and polish (Blue Letter Bible, various denominational apps). YouVersion reaches a wide audience but has no academic layer. Sefaria covers Jewish texts well but nothing beyond. No single free platform handles the breadth of traditions with the depth serious students need.

This organization exists to close that gap.

---

## Principles

- **No paywall to read.** Reading and browsing require no account. Authentication only unlocks personal features (cross-device sync, annotations, study plans).
- **Open corpus.** Only public domain and permissively licensed translations are used. Every text source is documented and auditable.
- **Reliable tools.** Software that works, documented clearly, maintained consistently.
- **Knowledge shared freely.** These tools are free because the knowledge they deliver should be.

---

## Active Projects

### [soltana](https://github.com/soltana-tech/soltana)

Academic scripture study platform. Currently in pre-release (target: January 1, 2027).

Covers Abrahamic traditions first — Christian, Jewish, Islamic — with canonical and extracanonical texts. The architecture supports expansion into Ancient Near Eastern, Zoroastrian, Buddhist, and Hindu traditions in later phases. Only open/public domain translations are included.

Features in scope: scripture reader, word study (Strong's, morphology), annotations and notebooks, custom study plans, historical timeline, full-text and semantic search, and a lesson/sermon builder.
