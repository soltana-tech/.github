## Why This Exists

Free, serious study tools for religious texts are genuinely scarce. Software with real academic depth — original languages, morphology, cross-references, comparative traditions — either costs significant money or lacks the breadth and polish that serious students need. No single free platform handles the range of traditions at a level worth relying on.

The goal of this organization is to close that gap with software that is reliable, easy to work with, and openly maintained.

## Principles

- **Knowledge shared freely.** Depth of access to primary religious texts should not require a subscription or cost.
- **No account required to read.** Authentication only unlocks personal features — cross-device sync, annotations, and study plans.
- **Open corpus.** Only public domain and permissively licensed translations are used. Every text source is documented and auditable.
- **Reliable and maintainable.** Software that works, documented clearly, and kept consistent over time.

## Active Project

### [Soltana](https://github.com/soltana-tech/soltana)

An open-source academic scripture study platform. Near-term focus is Abrahamic traditions — Christian, Jewish, and Islamic — with canonical and extracanonical coverage. The architecture supports expansion into Ancient Near Eastern, Zoroastrian, Buddhist, and Hindu traditions without structural rework.

Features in scope: scripture reader, word study (Strong's, morphology), annotations and notebooks, custom study plans, historical timeline, full-text and semantic search, and a lesson/sermon builder. Only open and public domain translations are included.

> [!NOTE]
> Soltana is currently in pre-release. Initial release target is January 1, 2027.

See the [repository](https://github.com/soltana-tech/soltana) for the development roadmap, architecture documentation, and contribution guidelines.
